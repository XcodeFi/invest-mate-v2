## Context

Hệ thống hiện tại thiếu khâu **đóng chiến dịch** — mỗi TradePlan khi kết thúc chỉ flip status sang `Reviewed` mà **không lưu bất kỳ dữ liệu đánh giá nào**. Không có cách so sánh hiệu quả giữa các kế hoạch, không có vòng lặp rút kinh nghiệm.

### Hiện trạng

| Thành phần | Trạng thái | Ghi chú |
|---|:---:|---|
| `MarkReviewed()` | ✅ | Chỉ flip status, không lưu data |
| P1 Post-Trade Review | ✅ | Đánh giá từng **trade đơn lẻ**, không phải toàn bộ kế hoạch |
| AI Trade-Plan-Advisor | ✅ | Đánh giá kế hoạch **trước** khi thực thi |
| Đánh giá **sau** chiến dịch | ❌ | **Không có** |
| So sánh hiệu quả giữa các KH | ❌ | **Không có** |
| TimeHorizon trên TradePlan | ❌ | **Không có** — cần thêm để phân loại |

### Yêu cầu

- Scoring = **% lợi nhuận và số tiền lãi/lỗ thực tế** — không chấm điểm chủ quan
- Chuẩn hóa về **hiệu suất/ngày** (VND/ngày, %/ngày) để so sánh công bằng giữa chiến dịch dài ngắn khác nhau
- **Chỉ so sánh cùng loại** — ngắn hạn vs ngắn hạn, dài hạn vs dài hạn
- Bài học rút kinh nghiệm sau mỗi chiến dịch
- Tổng hợp cross-plan analytics để tracking cải thiện qua thời gian

### Luồng

```
Lập KH → Thực thi → Đóng chiến dịch (auto-tính metrics) → Ghi bài học
                                    ↓
              Tích lũy → So sánh cùng loại → Cải thiện KH sau
```

**1d. Sửa `MarkReviewed()` — nhận review data thay vì empty:**
```csharp
public void MarkReviewed(CampaignReviewData reviewData)
{
    if (Status != TradePlanStatus.Executed)
        throw new InvalidOperationException("Only executed plans can be reviewed");
    ReviewData = reviewData ?? throw new ArgumentNullException(nameof(reviewData));
    Status = TradePlanStatus.Reviewed;
    UpdatedAt = DateTime.UtcNow;
    IncrementVersion();
    AddDomainEvent(new PlanReviewedEvent(Id, UserId, reviewData.PnLPercent));
}
```

### Công thức tính (tất cả auto từ trades liên kết):

| Metric | Công thức | Nguồn |
|---|---|---|
| TotalInvested | `SUM(qty × price + fee + tax)` cho BUY trades | `Trade[]` linked to plan |
| TotalReturned | `SUM(qty × price - fee - tax)` cho SELL trades | `Trade[]` linked to plan |
| PnLAmount | `TotalReturned - TotalInvested` | Calculated |
| PnLPercent | `PnLAmount / TotalInvested × 100` | Calculated |
| HoldingDays | `(lastTradeDate - firstTradeDate).Days`, min 1 | `Trade.TradeDate` |
| **PnLPerDay** | `PnLAmount / HoldingDays` | **Thước đo so sánh chính** |
| AnnualizedReturn | `((1 + PnL%/100)^(365/days) - 1) × 100` | Calculated |
| TargetAchievement | `actualPnL% / plannedTarget% × 100` | plan.Target vs actual |

## 3. Application Service — CampaignReviewService

### New interface: `src/InvestmentApp.Application/Common/Interfaces/ICampaignReviewService.cs`

```csharp
public interface ICampaignReviewService
{
    Task<CampaignReviewMetrics> CalculateMetricsAsync(TradePlan plan, IEnumerable<Trade> trades, CancellationToken ct = default);
}
```

### Implementation: `src/InvestmentApp.Infrastructure/Services/CampaignReviewService.cs`

Logic:
1. Tách BUY/SELL trades
2. Tính TotalInvested, TotalReturned, TotalFees
3. PnLAmount = TotalReturned - TotalInvested
4. HoldingDays = (last - first trade date), min 1
5. PnLPerDay = PnLAmount / HoldingDays
6. AnnualizedReturn — cap hiển thị nếu holding < 7 ngày (tránh số cực lớn vô nghĩa)
7. TargetAchievement = actualPnL% / ((Target - Entry) / Entry × 100) × 100

## 4. CQRS Commands & Queries

### Commands:

| Command | File | Mô tả |
|---|---|---|
| `ReviewTradePlanCommand` | `Application/TradePlans/Commands/ReviewTradePlan/` | Đóng chiến dịch: auto-tính metrics + lưu lessons |
| `UpdateReviewLessonsCommand` | `Application/TradePlans/Commands/UpdateReviewLessons/` | Cập nhật bài học sau khi đã review |

**ReviewTradePlanCommand handler flow:**
1. Load plan → verify ownership + status == Executed
2. Collect trades: từ `plan.TradeIds` + `ITradeRepository.GetByTradePlanIdAsync()` → deduplicate
3. `ICampaignReviewService.CalculateMetricsAsync(plan, trades)`
4. Build `CampaignReviewData` (metrics + user's LessonsLearned)
5. `plan.MarkReviewed(reviewData)`
6. Save → Return `CampaignReviewDto`

**Sửa `UpdateTradePlanStatusCommand`:** Bỏ case `"reviewed"` — buộc dùng `ReviewTradePlanCommand` (không thể review mà thiếu data)
