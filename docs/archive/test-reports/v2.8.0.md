### Bugs từ Round 8 (chưa fix)

| ID | Severity | Mô tả |
|---|---|---|
| **M4** | Medium | "Bán 50%" button không auto-fill Số lượng/Giá |
| **M7** | Medium | Wizard ghi 150 CP thay vì 100 CP (Chia lô 50+50) |
| **L7** | Low | Journal entry EVF mồ côi (từ dữ liệu cũ) |

### Root Cause Analysis — M9 & M10
Cả hai bugs có chung nguyên nhân: **Angular route không đọc query params khi component init**. Trade Plan component chỉ load data khi user click row, không subscribe vào `ActivatedRoute.queryParams` để auto-populate form. Cần thêm logic trong `ngOnInit()`:
```typescript
this.route.queryParams.subscribe(params => {
  if (params['loadPlan']) this.loadPlanById(params['loadPlan']);
  if (params['symbol']) this.form.patchValue({ symbol: params['symbol'], ... });
});
```

## 6. ĐIỂM NỔI BẬT CỦA ỨNG DỤNG

Qua 8 vòng testing, Investment Mate v2 nổi bật ở:

1. **Risk Management** là module tốt nhất — 4 preset Risk Profiles, Position Sizing calculator, Checklist 13 items (PHÂN TÍCH / QUẢN LÝ RỦI RO / TÂM LÝ / XÁC NHẬN), SL/TP management tập trung. Đây là giá trị cốt lõi differentiator so với các app trading khác.

2. **Companion App Concept** — Plan → Execute on broker → Log back. Không cố cạnh tranh với broker mà BỔ SUNG. Triết lý đúng.

3. **DCA Mode** — Rất ít app có DCA mode tích hợp với Trade Plan. UI riêng biệt, chuyên nghiệp.

4. **Phiếu lệnh** — Tính năng unique: tạo order sheet dạng text, copy/print để nhập vào broker. Thực tế và hữu ích.

5. **Live Market Data** — Đã fetch giá real-time ở một số trang (Dashboard, Analytics, Positions). Foundation tốt cho API tự động.
